PSC:
	ADD <num1> <num2> <num3>
		Assigns num1 + num2 to num3.
	SUBTRACT <num1> <num2> <num3>
		Assigns num1 - num2 to num3.
	MULTIPLY <num1> <num2> <num3>
		Assigns num1 * num2 to num3.
	DIVIDE <num1> <num2> <num3>
		Assigns num1 / num2 to num3.
	EXPONENT <num1> <num2> <num3>
		Assigns num1 ^ num2 to num3.
	BEEP
		Makes a noise.
	CLEARSCREEN
		Clears the console.
	COMMENT <comment>
		Comments a line.
	CONCAT <var1> <var2> <var3>
		Assigns var1 + var2 to var3.
	COPY <var1> <var2>
		Assigns var1 to var2.
	DIR <folder>
		Changes the working directory.
	EXIT <exitcode>
		Closes this instance.
	LOG <message>
		Outputs a message to a log file.
	LOGVAR <variable>
		Outputs a variable to a log file.
	MAKEFOLDER <folder>
		Creates a folder.
	OUTPUT <message>
		Outputs a message to the console.
	OUTPUTVAR <variable>
		Outuputs a variable to the console.
	READ <variable>
		Reads from the console into a variable.
	SETVAR <variable> <value>
		Sets a variable.
	STARTPROCESS <process>
		Starts a process.
	TONUMBER <strvar> <numvar>
		Converts a string to a number.
	TOSTRING <numvar> <strvar>
		Converts a number to a string.
